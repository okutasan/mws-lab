Google mengadakan pelatihan Mobile Web Specialist di Semarang
Training
Web
Google
Semarang

Training Mobile Web Specialist ini akan membimbing sekaligus menguji keterampilan kalian dalam mengembangkan aplikasi mobile web.

Kalian akan dipandu untuk menulis kode guna mempelajari dan memahami konsep dan implementasi pengembangan Mobile berbasis Web antara lain dengan mampu membuat aplikasi Off-Line, bekerja dengan Service Worker, memahami cara kerja proses asynchronous, dan masih banyak lagi fitur lainnya.

Pada akhir training kalian akan siap untuk mengikuti Google Certified Mobile Web Specialist, yang akan membedakan kalian dengan pengembang Web lainnya.

Metode belajar program berbasis kolaborasi dan komunitas ini menggunakan materi online yang dipadukan dengan kelompok belajar secara offline (study group).
